MM Breakpoints

Media-query width and height Sass mixin

This mixin is perfect to use in your Bootstrap project. It is using the default sizes

  
  @import 'm.breakpoints';

  .col {
    width: 100%;
    @include at-small  { width: 50% }
    @include at-medium { width: 33% }
    @include at-large  { width: 25% }
  }

You are in a none xsmall small medium large x none xsmall small medium large breakpoints.

You can see this only in xsmall small medium large.
